# Custom Commands for Hackathon AI Projects

## Quick Setup Commands
- `setup-dev-env`: Initialize a new development environment with common tools
- `create-component`: Generate a new component with boilerplate code
- `add-api-integration`: Add integration with common APIs (OpenAI, etc.)

## Development Commands
- `run-tests`: Execute project tests with coverage report
- `start-dev-server`: Launch development server with hot reloading
- `build-prod`: Create production build of the application

## Hackathon-Specific Commands
- `time-check`: Display remaining time in hackathon (when known)
- `idea-brainstorm`: Generate ideas based on current project context
- `demo-prep`: Prepare application for demo/presentation mode
- `quick-deploy`: Deploy to common hosting platforms (Vercel, Netlify, Heroku)

## Debugging Commands
- `debug-local`: Set up local debugging environment
- `log-analyzer`: Analyze logs and suggest fixes
- `performance-check`: Identify performance bottlenecks

## Code Quality Commands
- `lint-fix`: Apply automatic code formatting and linting fixes
- `security-scan`: Check for common security vulnerabilities
- `dependency-update`: Update dependencies to latest compatible versions
