## About This Textbook

This textbook follows the core principles of Physical AI education:
1. Specification-First Thinking
2. Hands-On, Simulation-First Learning
3. AI as a Co-Creator
4. Progressive Skill Building
5. Practical System Design

Each lesson follows a consistent structure:
- Learning Objectives
- Overview
- Theory
- Hands-On / Exercise
- Summary
- References / Resources

The content is designed to be modular and RAG-friendly, with beginner-friendly language and minimal jargon. All lessons emphasize simulation-first approaches to make Physical AI education accessible without requiring expensive hardware.
